IDE Shortcuts (Windows)

Task Android Studio Eclipse
go to declaration Ctrl + B F3
resolve error Alt + Enter F2
previous code tab Alt + ← Alt + Page Up
next code tab Alt + → Alt + Page Down
format code (spacing, line breaks, etc) Ctrl + Alt + L Ctrl + Shift + F
collapse block Ctrl + - Ctrl + -
expand block Ctrl + + Ctrl + +
collapse all blocks Ctrl + Shift + - Ctrl + Shift + /
expand all blocks Ctrl + Shift + + Ctrl + Shift + *
cut line (no need to select line of code) Ctrl + X
copy line (no need to select line of code) Ctrl + C
duplicate line to below (no need to select line of code) Ctrl + D Ctrl + Alt + ↓
duplicate line to above (no need to select line of code) Ctrl + Alt + ↑
single line comment/uncomment Ctrl + / Ctrl + /
multiple line (or any highlighted text) comment Ctrl + Shift + / Ctrl + Shift + /
multiple line (or any highlighted text) uncomment Ctrl + Shift + / Ctrl + Shift + \
close tab Ctrl + F4 Ctrl + W
search entire project(s) Ctrl + Shift + F Ctrl + H
show method parameters Ctrl + P F2
